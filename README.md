# SHSHchecker

Usage:

    make                  # Compile the program
  
    ./SHSHchecker <ECID> <MODEL>
  
example: 
              
    ./SHSHchecker 2339416931829 iPhone4,1

Arguments:
  <ECID>               ECID of the device ( only hexadecimal)
  <MODEL>              Device identifier (e.g., iPhone2,1)
